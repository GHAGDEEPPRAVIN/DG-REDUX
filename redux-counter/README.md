<h1>🔢 Counter App</h1>

A simple and responsive counter application built with HTML, CSS, and Redux Toolkit ⚡.
It demonstrates how to manage state using a centralized Redux store, structured state, and a clean slice setup.

<h3>✨ Features:</h3>

➕ Increment – Increase the counter value.

➖ Decrement – Decrease the counter value.

🔄 Reset – Set the counter back to zero.

🗂 Centralized Store – State is managed in one predictable location.

🧩 Slice Structure – Reducers + actions in a single clean file.

🔍 Debug Friendly – Works perfectly with Redux DevTools.

<h3>🛠 Tech Stack:</h3>

🖥 Frontend: HTML, CSS

🗄 State Management: Redux + Redux Toolkit (createSlice, configureStore)

⚙ Core Concepts: Store, State, Actions, Reducers
